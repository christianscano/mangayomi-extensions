# Mangayomi Extensions

This repository contains the available extension catalogues for the Mangayomi app.

## How to add the extensions

Add them manually in the app:

manga repo

```url
https://christianscano.github.io/mangayomi-extensions/index.json
```

anime repo

```url
https://christianscano.github.io/mangayomi-extensions/anime_index.json
```

novel repo

```url
https://christianscano.github.io/mangayomi-extensions/novel_index.json
```
